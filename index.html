pertemuan 2: Machine Learning Lifecycle

Adalah sebuah workflow atau alur kerja pengembangan model machine learning. machine learning lifecycle bersifat forward (maju), dan dapat berulang (iteratif) karena bertujuan untuk terus meningkatkan keakuratan dan performa model
  • Machine Learning Workflow
  1. pengumpulan data (data aquasition)
    tipe artibut datanya berupa:
    • data nominal
    • data ordinal
    • data interval
    • data ratio
  2. persiapan data
    • pembersihan data (data cleaning): melakukan operasi perbaikan pada level baris.

      • pembersihan data atau data cleaning adalah proses mendeteksi dan memperbaiki (atau menghapus) data yang rusak atau tidak akurat dari dataset,tabel, atau data untuk meningkatkan kualitas data
      langkah-langkah pembersihan data:
      1. pendeteksian data yang tidak diharapkan
        teknik menemukan error pada data:
        • data profiling
        • visualisasi data
      2. data cleaning, penghapusan, perbaikan/penghitungan ulang.
        ketika ada pada kondisi:
        • data yang tidak relevan
        • data yang terduplikasi
        • konversi tipe data
        • perbaikan dan modifikasi data
        • standarisasi data
        • perbaikan missing data
        • transformasi data
        • normalisasi data
    • rekayasa fitur: perbaikan data pada level kolom

pertemuan 6: Rekayasa Fitur

Rekayasa fitur adalah proses perubahan datset di mana dilakukannya proses ekstrasi sebuah kolom menjadi kolom-kolom lainnya/proses mereduksi kolom.
  • kriteria fitur yang baik:
    1. informatif
    2. ketersediaan data
    3. diskriminan
  • metode dalam proses rekayasa fitur:
    1. correlation matrix: memiliki korelasi yang tinggi dengan cariabel dependent
    2. univariate selection: menggunakan fungsi test statistik dalam menentukan seberapa penting kolom
    3. menggunakan midel yang berbasis tree
    4. permutation feature importance
    5. feature importance

pertemuan 2

Citra adalah suatu gambaran, kemiripan, atau imitasi dari suatu objek fisik manusia
Citra digital juga didefinisikan sebagai fungsi f(x,y)
  1. Citra Analog, bersifat fisik. contoh: monitor televisi, foto sinar x, lukisan, hasil CT scan
  2. Citra Digital, merupakan representasi dari citra yang diambil oleh mesin dengan bentuk pendekatan berdasarkan sampling dan kuantisasi
  • jenis citra digital:
    1. Citra biner (monokrom): dua warna yaitu hitam dan putih
    2. Citra grayscale (skala keabuan): banyaknya warna tergantung pada jumlah
    3. Citra warna (true color): red, green, blue
  • elemen-elemen citra digital:
    1. kecerahan
    2. kontras
    3. kontur
    4. warna
    5. bentuk:
      • 2 dimensi
      • 3 dimensi
    6. tekstur: keteraturan pola-pola tertentu dari susunan pixel dalam citra digital.

pertemuan 1: 9 September 2024

Definisi Sistem Informasi Manajemen

Sistem Informasi manajemen merupakan mata kuliah yang akan membahas mengenai pengolahan data informasi terkait manajemen

pertemuan 2: 18 September 2024

Sistem Informasi Manajemen adalah metode yang menggabungkan dan menciptakan informasi yang solid
  • konsep dasar SIM:
    1. pengumpulan data
    2. pengolahan data
    3. penyimpanan data
    4. distribusi informasi: menyebarkan hasil data yang telah dikelola
    5. pengambilan keputusan: harus bisa mengatasi masalah
    6. kontrol dan pemantauan: pengecekan kembali pada data
    7. integrasi sistem: harus bisa mengklasidikasikan datanya menggunakan sistem
    8. keamanan dan privasi

pertemuan 5: 9 Oktober 2024

Perancangan System Input Output dan Manajemen Operasi

Perancangan Input adalah proses merancang dan menentukan bagaimana data / informasi akan dimasukan ke dalam sistem / aplikasi. Perancangan input berfungsi untuk meningkatkan interaksi pengguna dengan sistem dan memastikan data yang dihasilkan berkualitas tinggi.
